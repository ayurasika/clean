カテゴリー別マイクロ指示アルゴリズムの設計と実装

本セクションでは、具体的なアイテムカテゴリーに対し、30秒から2分で完了し、かつ視覚的な「秩序感」の向上が著しいアクションを定義する。これらの指示は、ゲシュタルト心理学の諸法則を応用し、脳が「片付いた」と認識しやすいパターンの創出を目的としている。

書類と紙類のマイクロアルゴリズム

書類は情報の密度が高く、一枚一枚の内容を確認しようとすると即座に認知オーバーロードを引き起こす。そのため、指示は「内容の解読」を徹底的に排除し、「幾何学的な整合」に特化させる。

マイクロ指示フレーズ適用されるゲシュタルト法則所要時間視覚的変化の根拠「バラバラの紙を1つの束にまとめ、角を机の角に合わせる」共通運命の法則 

30秒輪郭が明確になり、空間内のノイズが低減される。「DMやチラシを、大きいものから順に重ね直す」類同の法則 

60秒形状の連続性が生まれ、複数の物体が「1つの構造物」として認識される。「クリップやクリアファイルに入った書類を、背表紙を揃えて立てる」近接の法則 

90秒垂直方向への秩序が生まれ、床や机の設置面積（占有率）が減少したように見える。

衣類と布製品のマイクロアルゴリズム

衣類は不定形であるため、視覚的な混乱を招きやすい。指示の鍵は「境界の確定」と「向きの統一」である。

境界の確定: 床に散らばった衣類を「一旦ベッドの上に置く」あるいは「カゴに入れる」という行為は、ゲシュタルトの「閉合の要因」を利用している 。空間全体に拡散していた無秩序が、カゴという境界線の中に閉じ込められることで、脳は「床が綺麗になった」というポジティブな信号を受け取る。

向きの統一: ハンガーに掛かっている衣類の「向き」を揃える。これは「類同の法則」および「共通運命の法則」に基づき、個別の衣類を「一貫したパターン」へと変換する 。

食器およびキッチン用品のマイクロアルゴリズム

キッチンは蛇口やコンロなどの固定設備が存在するため、それらとの「位置関係」を整理することが重要である。

平行と垂直の維持: シンクの中にある食器を、重ねられるもの同士で積み上げる。これにより「近接の法則」が働き、バラバラの物体が「ユニット」化される 。

素材によるグルーピング: ガラス製品、陶器、プラスチックなど、素材（＝光沢や質感）が似ているものを集める。これは「類同の法則」に基づき、視覚的なテクスチャの乱れを整える効果がある 。

文房具とおもちゃのマイクロアルゴリズム

これらはサイズが小さく、数が多い。そのため「点」として散在する情報を「面」または「塊」へと統合する指示が有効である。

色による分類: 「赤いものだけをペン立てに戻す」といった指示は、類同の法則の中でも特に強い「色彩の類似性」を利用する 。検索コストを下げつつ、短時間で「一画が整理された」という実感を創出する。

中心への集約: 部屋の隅々に散らばった小さなおもちゃを、部屋の中央にあるラグや特定のボックスへ「とりあえず集める」。これは「近接の要因」を極大化させ、視覚的な「ノイズの局所化」を実現する 。
